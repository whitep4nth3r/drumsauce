# DRUMSAUCE

Browse free PDF downloads of rock, metal and jazz music transcriptions for drummers. Created by human ears and hands, no AI involved.

Visit drumsauce at [drumsauce.net](https://drumsauce.net)

Subscribe to the DRUMSAUCE RSS feed at [drumsauce.net/rss.xml](https://drumsauce.net/rss.xml)
